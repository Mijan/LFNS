16  * This is the header file for a generic matrix package.
17  * It defines the SUNMatrix structure (_generic_SUNMatrix) which
18  * contains the following fields:
19  * - an implementation-dependent 'content' field which contains
20  * the description and actual data of the matrix
21  * - an 'ops' filed which contains a structure listing operations
22  * acting on such matrices
23  * -----------------------------------------------------------------
24  * This header file contains:
25  * - enumeration constants for all SUNDIALS-defined matrix types,
26  * as well as a generic type for user-supplied matrix types,
27  * - type declarations for the _generic_SUNMatrix and
28  * _generic_SUNMatrix_Ops structures, as well as references to
29  * pointers to such structures (SUNMatrix), and
30  * - prototypes for the matrix functions which operate on
31  * SUNMatrix objects.
32  * -----------------------------------------------------------------
33  * At a minimum, a particular implementation of a SUNMatrix must
34  * do the following:
35  * - specify the 'content' field of SUNMatrix,
36  * - implement the operations on those SUNMatrix objects,
37  * - provide a constructor routine for new SUNMatrix objects
38  *
39  * Additionally, a SUNMatrix implementation may provide the following:
40  * - macros to access the underlying SUNMatrix data
41  * - a routine to print the content of a SUNMatrix
42  * -----------------------------------------------------------------*/
