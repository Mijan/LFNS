\doxysubsection{Detailed Description}
This is the common ancestor class for the sequential (L\+F\+N\+S\+Seq) and parallel (L\+F\+N\+S\+Mpi) \mbox{\hyperlink{classlfns_1_1LFNS}{L\+F\+NS}} algorithms. It is based on the publication Mikelson, Jan, and Mustafa Khammash. \char`\"{}\+Likelihood-\/free nested sampling for biochemical
reaction networks.\char`\"{} bio\+Rxiv (2019)\+: 564047. See this publication for all details on the functioning ond parameters of this algorithm
