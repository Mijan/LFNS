\doxysubsection{Detailed Description}
This class implements the input pulses for different experiments. Each \mbox{\hyperlink{classmodels_1_1InputPulse}{Input\+Pulse}} contains information about the input parameter, which is the parameter to be modified by the \mbox{\hyperlink{classmodels_1_1InputPulse}{Input\+Pulse}} and information about the time during which the pulse should be applied. It also containes the value of the input.
