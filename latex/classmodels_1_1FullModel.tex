\doxysubsection{Detailed Description}
The class representing the full model that can be simulated using one of the simulators (currently O\+DE simulator S\+SA simulator and a hybrid simulator. This class provides a way to compute the initial state of a system, to evaluate the dynamics for an o\+DE simulator and compute the propensity for the S\+SA simulator. It also provides a way to simulate a measurement at any given timepoint, which is considered to not be a part of the model dynamics. This class also allows to apply inputs to the model dynamics to simulate various experiments.
